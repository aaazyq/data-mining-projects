3.2 排量情况和保值比率的关系

dat1$pailiangqingkuang <- as.character(" ")

dat1$pailiangqingkuang[dat1$pailiang < 1.0] <-"微型轿车"
dat1$pailiangqingkuang[dat1$pailiang < 1.6 & dat1$pailiang >=1.0] <-"普通级轿车"
dat1$pailiangqingkuang[dat1$pailiang < 2.5 & dat1$pailiang >=1.6] <-"中型轿车"
dat1$pailiangqingkuang[dat1$pailiang < 4.0 & dat1$pailiang >=2.5] <-"中高级轿车"
dat1$pailiangqingkuang[dat1$pailiang >= 4.0] <-"高级轿车"
dat1$pailiangqingkuang <- as.factor(dat1$pailiangqingkuang)

l <- c("微型轿车","普通级轿车","中型轿车","中高级轿车","高级轿车")
dat1$pailiangqingkuang = factor(dat1$pailiangqingkuang,levels = l)
boxplot(Logbazhibilv ~pailiangqingkuang ,data = dat1,

col="orange",
ylab="Log-保值比率",
main="排量情况和log-保值比率的关系")

可以看出，随着排量上升，保值率先上升再下降——也就是说排量过低或排量过高都不利于汽车保值。其中，普通型轿车的保值率最高，微型轿车保值率之间差异最大。
